Portafolio de Prácticas

Ejercicio Día 2

Creación de una página web estática sobre películas, utilizando HTML y CSS. Incluye un menú de navegación con géneros, una lista ordenada de un top 10 de películas y una imagen codificada en base64 para la película 'Cars' con una breve descripción.

Práctica 1

Creación de una página personal estática con HTML y CSS, presentando mi perfil, hobbies, habilidades y enlaces a redes sociales, con un diseño simple y estructurado.
Práctica 2

Desarrollo de un formulario de inicio de sesión con HTML y CSS. Incluye campos para email y contraseña, una casilla de verificación, enlaces a redes sociales y un diseño responsivo con estilos personalizados.
Práctica 3

Creación de una página web estática con HTML y CSS, que incluye un encabezado, un menú de navegación, una sección hero con imagen y botón, una lista de elementos y un pie de página simple.
Práctica 5

Calculadora de áreas geométricas desarrollada en JavaScript. Permite calcular el área de un cuadrado, rectángulo o triángulo mediante un menú interactivo con validación de entradas, usando funciones y estructuras de control.
Práctica 6

Simulador de cuenta bancaria en JavaScript que permite depositar, retirar y consultar saldo mediante un objeto con métodos. Utiliza un menú interactivo con prompts para la entrada de datos y alertas para mostrar resultados.
Práctica 7

Manipulación de Arrays en JavaScript "Uso de métodos de arrays en JavaScript (forEach, map, includes, filter, slice, sort, reverse) para gestionar un listado de productos. Incluye operaciones como mostrar productos, aplicar descuentos, filtrar por precio y ordenar datos, presentados en una interfaz web responsiva.

Práctica 8

Diseño de una interfaz estática para un juego de Tic Tac Toe utilizando HTML y Tailwind CSS. Incluye un tablero interactivo, indicadores de turno y un panel de puntajes con un diseño responsivo y moderno.

Práctica 9

Desarrollo de un juego interactivo de Tic Tac Toe usando HTML, Tailwind CSS y JavaScript. Incluye una cuadrícula de 3x3, seguimiento de turnos, puntajes para jugadores (X, O y empates), y modales para mostrar resultados y reiniciar el juego.
Práctica 10

Desarrollo de una página web que consume datos de una API pública para mostrar una lista de usuarios en tarjetas. Utiliza HTML, Tailwind CSS para estilos responsivos y JavaScript con fetch para manejar solicitudes asíncronas, incluyendo gestión de errores.
